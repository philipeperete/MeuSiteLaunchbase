## 🚀 Como executar o projeto

```bash
# Clone este repositório.
$ git clone https://github.com/philipeperete/MeuSiteLaunchbase

# Acesse a pasta onde está o arquivo "server.js" do projeto no terminal/cmd.
$ node server.js

# Abre o navegador e acesse: 
$ http://localhost:5000/

```
